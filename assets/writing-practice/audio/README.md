# Edmund Neural writing audio

These MP3 files are build-time outputs for the English essays in
`writing-practice.html`.

- Model: Kokoro-82M v1.0 (Apache-2.0), hexgrad
- Runtime: kokoro-onnx 0.5.0 (MIT), thewh1teagle
- Voice: `af_heart`
- Language: `en-us`
- Speed: `0.96`
- Format: mono 24 kHz MP3

Rebuild and validation instructions are in
`tools/EDMUND-NEURAL-VOICE-SOP.md`.
